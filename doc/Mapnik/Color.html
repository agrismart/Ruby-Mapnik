Class Method Details

+ (Mapnik::Color) new + (Mapnik::Color) new(red, green, blue) + (Mapnik::Color) new(red, green, blues, alpha) + (Mapnik::Color) new(css_string)

Returns a new Color object. The first form returns a value of white (255, 255, 255, 255), the others set the values properly.

The CSS String form should be of the format “#FF0000” (leading ‘#’).
